Will any animals coming on to this protocol be classed as ‘continued use’?
‘Continued use’ describes animals that are specifically genetically altered and bred for scientific use or animals that have had procedures applied to them in order to be prepared for use in this protocol.

Will you be re-using animals on to this protocol?
‘Re-use’ describes using animals again for a new experiment when you could equally use a naïve animal to get the same results.

For example, if some animals will go through this protocol three more times after their first use, the number of uses will be four.

If no animals will go through this protocol more than once, enter '1'.

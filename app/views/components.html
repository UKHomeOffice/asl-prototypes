Application/licence status banners

Displayed at the top of applications or licences that are not currently active to indicate to users that science is not authorised under them.

Not-yet-active banners: e.g. draft

When an application has not yet been granted we use the grey banner. This is used for the 'draft' or 'amendment in progress' states to be clear that the document the user is working on has not yet been granted.

{% include "includes/status-banner-draft.html" %}

No-longer-active banners: e.g. dxpired

When a licence has at one point, but is no longer, active we use the red banner. This is used for the 'expired', 'revoked', 'suspended', 'superseeded', 'transfered out' or had it's 'additional availability removed'. These will also contain the grant date and date the licence became inactive.

{% include "includes/status-banner-expired.html" %}

Toaster messages

Used to confirm actions by the user when they haven't left the page the action was taken on. Thes should be the full width of the window, stick to the top of the window and fade out after 5 seconds.

{% include "includes/toaster-example.html" %}

In-system notifications

Displayed on external users home pages and relevant licences to warn them of upcoming required actions. These will appear when the user has outstanding returns of procedures, retrospective assessments, 5 year PIL reviews or licence conditions requiring action.

Basic user

Basic users will only receive these notifications intermitently, so each notification has a unique banner. {% include "includes/in-system-notification-basic.html" %}

Admin user

Admin users receive notifications for all users at their establishments. Therefore they get a consolidated banner for all the notifications at each establishment. They will also receive individual notifications for actions related to licences they hold personally.

{% include "includes/in-system-notification-holc.html" %}

Conditions

Conditions are added to each of the licence types by the inspector assessing them. They are displayed as per the GOV.UK inset text but using the Home Office purple, so they stand out from other content on the page

Status badges

Used typically in table views to highlight the status of the items in the list. There are a range of different labels depending on the context but the colours used are as follows:

Grey: The thing is in a nutural state, e.g. awaiting a decision or not yet active

Awaiting decision Draft Incomplete

Blue: The thing is in a positive state, e.g. approved or currently active

Approved  Active Complete

Red: The thing is in a negative state, e.g. rejected or expired

Rejected  Expired  Revoked
